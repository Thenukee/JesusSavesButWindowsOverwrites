## Overview

Every researcher and developer needs a home on the web. This site is mine — built from scratch with a focus on speed, readability, and a clean aesthetic.

## Design Principles

- **Speed** — Static HTML, minimal JavaScript, no heavy frameworks
- **Readability** — Clean typography, comfortable line lengths, proper contrast
- **Glassmorphism** — Subtle frosted glass effects for a modern look
- **Accessibility** — Semantic HTML, proper ARIA labels, keyboard navigation
- **Dark Mode** — Automatic detection with manual toggle

## Features

- Skeleton loading states for perceived performance
- Client-side blog search and tag filtering
- Auto-generated table of contents for long posts
- Reading time estimation
- Related posts suggestions
- RSS feed for subscribers
- Responsive design for all screen sizes
- SEO optimized with meta tags and structured data

## Tech Stack

The site is intentionally simple:

- **Jekyll** for static site generation
- **Vanilla CSS** with custom properties for theming
- **Vanilla JavaScript** for interactivity
- **GitHub Pages** for free hosting
- **Formspree** for the contact form

No build tools, no npm packages, no frameworks — just clean, fast HTML.

## Workflow

Writing a new post is simple:

1. Create a markdown file in `_posts/`
2. Add front matter (title, date, tags)
3. Write the content
4. Push to GitHub → published automatically
